# Hex Sovereign Architecture Pack

## Purpose

This pack turns the Hex Sovereign concept into an implementation-ready architecture plan for the new portfolio.

The first portfolio release should not attempt to ship the whole ruleset. It should prove the strongest technical idea:

> A deterministic static-browser strategy game with a validated legal-action protocol for bots, browser agents, and debugging tools.

## Architecture Decision

Build the engine and protocol first, then grow the game.

The game surface matters, but the portfolio differentiator is the clean boundary between:

- deterministic rules
- legal action generation
- validated action submission
- readable UI state
- browser-local AI integration
